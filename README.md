# Chatterbox TTS

Fork of [resemble-ai/chatterbox: SoTA open-source TTS](https://github.com/resemble-ai/chatterbox)

## Installation

```sh
# prod
uv sync --frozen --no-dev

# dev
uv sync --frozen
```

## Usage

### Run main script

```sh
uv run main.py
```

### Run examples

```sh
uv run examples/example_tts.py
```
